Prepare an input file outside of the app

If you choose to create the input file outside of the app (versus using the ‘File Builder’), the file needs to have all the required fields listed in the tables below. Column names must match the spelling, symbology, and capitalization scheme exactly as shown.

IMPORTANT: Make sure you are using the proper Operational Taxonomic Unit (OTU) and taxa traits/attributes for your metric calculations. The Taxa Translator and Attribute tables (and associated metadata) that were used for P51 development can be downloaded: EGLE Taxa Translator Files.

The EGLE Input Test File can be used as a guide for your own input file: EGLE Test Files.

Input file requirements for ELGE P51 models

Column Name  Data Type Domain (allowable entries) Null Allowed Comment
SAMPLEID text NA No Unique sample identifier (typically comprised of the site name, sample date, and replicate number).
INDEX_CLASS text Narrow, VeryNarrow, MidSizeDry, WetWide, WestFlat, WestSteep, East No Stream class is assigned based on site characteristics. Can be manually chosen or chosen through the index classification app function (see File Builder).
TAXAID text NA No Taxonomic identification of macroinvertebrates in each sample.
N_TAXA numeric NA No Number of individuals.
NONTARGET text TRUE, FALSE No Non-target taxa are those that are not part of the intended capture list. These taxa will be removed before any calculations.
PHYLUM text NA Yes Entries should be consistent with the phylogeny.
SUBPHYLUM text NA Yes Entries should be consistent with the phylogeny.
CLASS text NA Yes Entries should be consistent with the phylogeny.
SUBCLASS text NA Yes Entries should be consistent with the phylogeny.
ORDER text NA Yes Entries should be consistent with the phylogeny.
FAMILY text NA Yes Entries should be consistent with the phylogeny.
FFG text CF; CG; PR; SC; SH Yes Functional feeding group. CF = “collector-filterer”; CG = “collector-gatherer”; PR = “predator”; SC = “scrapper”, and SH = “shredder”. More than one FFG can be assigned to a taxon - separated by “;” (e.g., “SH;SC”). Note, that other FFG designations are allowed, but are not used in the EGLE IBIs.
TOLVAL numeric NA Yes Taxa tolerance values. Range from 0 (intolerant) to 10 (tolerant).
HABIT text BU; CB; CN; SP; SW Yes Taxon habit. BU = “burrower”; CB = “climber”; CN = “clinger”; SP = “sprawler”, and SW = “swimmer”. More than one habit can be assigned to a taxon - separated by “;” (e.g., “CB;CN”).
