A single zip file will be available for download. It will include all result files, reference files, and the original input data file. All result data files will be CSV format, unless otherwise noted.

The zip file will included the base name of the original file along with the date and time.

The individual files and their contents are output in three subfolders. Each output file is described in the table below.

Calculation, zip file contents.
Output file Subfolder Description
Original file name _user_input original input file
MetricNames.xlsx reference Metrics calculated by the BioMonTools R package, their description, and input fields.
MetricScoring.xlsx reference Metric scoring table used by the BioMonTools R package.
IBI_1markexcl results_MI_EGLE_IBI Excluded (non-count) taxa. Excluded from taxa richness metrics. Calculated using BioMonTools R package (https://github.com/leppott/BioMonTools).
IBI_2metval_all results_MI_EGLE_IBI Values for all bug metrics in the BioMonTools R package (https://github.com/leppott/BioMonTools)
IBI_2metval_IBI results_MI_EGLE_IBI Values for bug metrics used in the MI EGLE IBI models; the calculations are done with the BioMonTools R package (https://github.com/leppott/BioMonTools)
IBI_3metscores results_MI_EGLE_IBI Scores (0-100) for bug metrics used in the MI EGLE IBI models.For each sample, IBI model output (metric values, metric scores, and index scores)
IBI_4Index_Attain results_MI_EGLE_IBI Attainment status of each sample based on MI EGLE thresholds.For each sample, IBI model output (metric values, metric scores, index scores, and attainment status)
